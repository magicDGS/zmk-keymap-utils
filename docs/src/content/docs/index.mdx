## Features


<CardGrid>
	<Card title="OS-suport" icon="laptop">
		Configure your keymaps for different operating systems, and build/share for different targes changing a single variable.
	</Card>
	<Card title="Shortcuts" icon="puzzle">
		Use consistent definitions for keycodes and shortcuts that just work on your OS.
	</Card>
</CardGrid>
<Card title="Popular implementations" icon="heart">
		Avoid copy-pasting popular implementations for ZMK keymaps by including header/dtsi files.
		Some examples include:
		- Select-Word macros based on the Sunaku's implementation of Pascal Getreuer's Select Word macro from QMK
		- Timeless home-row mods based on urob's implementation
	</Card>
